Shale SEM Image Classification and Segmentation

To start the neural network analysis of an SEM image, please begin by uploading a supported image file into the "File Upload" box below. After you finish uploading the SEM image below, you will be able to analyze the image for lithological classification as well as segmentation base on factors like Minerology, Porosity and/or Organic Matter. File Upload



Instructions

  1. Select the image file (file types here) by clicking "Query button" or by dragging the file into the File Upload box
  2. Once uploaded, please press "Process Image" to begin processing the file
  3. Once image has been processed, please move to either the "Classification" tab or "Segmentation" tab to continue the image analysis
